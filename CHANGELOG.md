# Learning Online Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-09

### Added

- Initial release
- Multi-language support (English, Japanese, Korean)
- Graded course system (A1-C2 levels)
- Interactive learning modules:
  - Vocabulary flashcards
  - Grammar exercises
  - Listening training
  - Speaking practice
- Learning progress tracking with daily goals and streak tracking
- User registration and login system
- Achievement and badge system
- Community features
- Responsive mobile-first design
- Modern flat UI design inspired by Google/Apple/Microsoft

### Tech Stack

- React 18 with Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Radix UI components
- Zustand for state management
- Prisma ORM with SQLite
- Lucide React icons
